# undistractify-stackoverflow
Undistractify Stackoverflow, Stackexchange, and all network sites


Removes distractions from:
- stackoverflow.com
- stackexchange.com
- superuser.com
- serverfault.com
- askubuntu.com
- stackapps.com
- mathoverflow.net
